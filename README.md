# Discord Bot for Web Scraping Financial Data

This Discord bot fetches financial data from a specified URL and sends it as a CSV file.

## Requirements

- Python 3.x
- Libraries:
  - requests
  - pandas
  - beautifulsoup4
  - discord.py

Install dependencies using:
```bash
pip install -r requirements.txt
